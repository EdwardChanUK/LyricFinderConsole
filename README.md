# LyricFinderConsole

Steps
1. Use Visual Studio 2022 to clone this repositary
2. Make sure installed .Net 6.0
3. Make sure the project LyricFinderConsole is the startup project
4. Rebuild solution
5. Start without debuging in Visual Studio
